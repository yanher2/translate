# 翻译应用功能说明

## 📱 应用概述

一个完全离线的英中翻译 Android 应用，使用 SQLite 词典数据库提供快速的翻译服务。

**应用包名：** `com.yhg.translate`
**版本：** 2.0
**词典词汇量：** 30万+

---

## ✨ 核心功能

### 1. 离线翻译

**功能描述：**
- 输入英文单词或短语，实时翻译为中文
- 完全离线工作，无需网络连接
- 支持常用词汇、短语、专业术语

**技术实现：**
- 使用 ECDICT SQLite 词典数据库
- 30万+ 词汇覆盖日常和学术需求
- 查询速度：毫秒级响应

**使用方法：**
1. 打开应用
2. 点击"翻译"按钮
3. 输入英文单词或短语
4. 查看翻译结果

**示例：**
- 输入：`hello` → 输出：`你好`
- 输入：`computer` → 输出：`计算机`
- 输入：`machine learning` → 输出：`机器学习`

---

### 2. 单词本

**功能描述：**
- 保存翻译结果到单词本
- 支持搜索、删除功能
- 数据持久化存储

**技术实现：**
- 使用 Room 数据库
- 存储位置：`/data/data/com.yhg.translate/databases/translate_database`
- 数据表：words (id, englishWord, chineseTranslation, timestamp)

**使用方法：**
1. 翻译某个单词后，点击"加入单词本"
2. 点击"单词本"查看保存的单词
3. 使用搜索框查找特定单词
4. 点击删除图标移除单词

**数据库结构：**
```sql
CREATE TABLE words (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    englishWord TEXT NOT NULL,
    chineseTranslation TEXT NOT NULL,
    timestamp INTEGER NOT NULL
);
```

---

### 3. 系统快捷翻译

**功能描述：**
- 在任何应用中长按选中文本
- 系统显示"翻译"选项
- 点击后跳转到本应用进行翻译

**技术实现：**
- Android PROCESS_TEXT Intent
- 在 `AndroidManifest.xml` 中注册
- 标签：`android:label="翻译"`

**使用方法：**
1. 在任何应用中长按选中文本
2. 在弹出的菜单中选择"翻译"
3. 自动跳转到翻译应用
4. 显示翻译结果

**注意事项：**
- 需要系统支持 PROCESS_TEXT
- 某些应用可能屏蔽此功能
- "翻译"选项按字母排序，靠前显示

### 4. 词典数据库

**功能描述：**
- 30万+ 英中双语词汇

**技术实现：**
- 数据库文件：`stardict.db` (811.85 MB)
- 数据来源：ECDICT (https://github.com/skywind3000/ECDICT)
- 数据库表：stardict

**词典信息：**

| pos | 词性 | int. |
| collins | 柯林斯星级 | 3 |
| oxford | 牛津频率 | 2 |
| tag | 标签 | c4 |
| bnc | 语料库频率 | 5243 |

**查询功能：**
- 精确匹配：查找完全匹配的单词
- 前缀搜索：查找以指定前缀开头的单词
- 关键词搜索：查找包含关键词的单词或翻译

```
│   └── WordEntity.kt                 # 单词本数据实体
├── network/
│   └── TranslationService.kt         # 翻译服务
├── ui/
│   └── screens/
│       ├── HomeScreen.kt             # 主界面
│       ├── TranslationBottomSheet.kt # 翻译底部面板
│       └── WordBookScreen.kt         # 单词本界面
└── viewmodel/
    ├── TranslationViewModel.kt       # 翻译视图模型
    └── WordBookViewModel.kt          # 单词本视图模型
```

## 💡 使用技巧

### 1. 提高翻译准确率

**技巧：**
- 输入单词原形（而不是复数或时态）
- 使用常见词汇，避免生僻词
- 短语翻译效果通常优于单词

**示例：**
- ✅ 输入：`run` → 翻译：`跑`
- ❌ 输入：`running` → 未找到（需要词形匹配）

### 2. 管理单词本

**技巧：**
- 定期整理单词本，删除不需要的单词
- 使用搜索功能快速查找单词
- 按时间排序查看最新添加的单词

### 3. 系统快捷翻译

**技巧：**
- 在浏览器、微信等应用中使用快捷翻译
- 长按选择文本时，确保文本完整
- 如果"翻译"选项不显示，检查系统设置

## ⚠️ 已知限制

### 1. 词汇覆盖

**限制：**
- 词典包含 30万+ 常用词汇
- 某些专业术语可能缺失
- 最新网络词汇可能未收录

**解决方案：**
- 尝试使用同义词
- 联系开发者添加词汇

### 2. 词形匹配

**限制：**
- 暂不支持复数、时态、比较级等词形变化
- 需要输入单词原形

**示例：**
- ❌ `books` → 未找到
- ✅ `book` → 找到：`书`

**未来计划：**
- 实现词形还原算法
- 支持常见的词形变化

### 3. APK 体积

**限制：**
- APK 包含 811.85 MB 词典数据库
- 安装包较大，下载时间较长

**未来计划：**
- 实现按需加载
- 分块下载词典
- 减少不必要的词汇

## 📞 技术支持

### 常见问题

**Q1: 为什么翻译某些单词显示"词典暂无此词"？**

A: 词典包含 30万+ 常用词汇，但可能不包含所有单词。请尝试：
1. 检查拼写是否正确
2. 使用单词原形（不是复数或时态）
3. 尝试使用同义词

**Q2: 首次启动为什么比较慢？**

A: 首次启动需要从 assets 复制 811.85 MB 的词典数据库到设备存储，需要 2-5 秒时间。

**Q3: 为什么在其他应用长按文本没有显示"翻译"选项？**

A: 这取决于系统和其他应用的设置。请尝试：
1. 确保已安装并打开本应用
2. 检查系统设置中的文本处理选项
3. 某些应用可能屏蔽了 PROCESS_TEXT 功能

**Q4: 单词本数据会丢失吗？**

A: 单词本数据存储在设备本地数据库中，只有在以下情况会丢失：
1. 卸载应用
2. 清除应用数据
3. 恢复出厂设置

**Q5: 如何备仢单词本？**

A: 当前版本不支持自动备份。未来版本将添加导出功能。
