## 🐛 问题排查

### 问题 1：应用启动缓慢

**可能原因：**
- 首次复制词典数据库（正常现象）
- 设备性能较低

**解决方案：**
- 等待 2-5 秒
- 检查存储空间是否充足（需要至少 2GB）

**验证：**
```bash
# 检查数据库复制进度
adb shell run-as com.yhg.translate ls -lh /data/data/com.yhg.translate/databases/
```

---

### 问题 2：翻译提示"词典暂无此词"

**可能原因：**
- 单词不在词典中
- 拼写错误
- 词形变化（如复数、时态）

**解决方案：**
1. 检查拼写是否正确
2. 尝试使用单词原形
3. 使用同义词

---

### 问题 3：单词本一直加载

**可能原因：**
- 数据库未正确初始化
- 之前的代码问题

**解决方案：**
- 已在 v2.0 中修复
- 验证数据库是否存在
- 清除应用数据重试

**验证：**
```bash
# 检查单词本数据库
adb shell run-as com.yhg.translate ls -lh /data/data/com.yhg.translate/databases/translate_database
```

---

### 问题 4：系统快捷翻译不显示

**可能原因：**
- 系统设置问题
- 其他应用屏蔽了 PROCESS_TEXT
- 应用未正确注册

**解决方案：**
1. 检查 AndroidManifest.xml 中的配置
2. 重启设备
3. 检查系统设置

**验证：**
```bash
# 查看应用配置
adb shell dumpsys package com.yhg.translate | grep -A 10 "android.intent.action.PROCESS_TEXT"
```
